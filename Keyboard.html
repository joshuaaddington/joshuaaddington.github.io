Split Keyboard

I designed and built a custom split mechanical keyboard using a hand-soldered logic matrix and an RP2040 microcontroller. The base is laser-cut and the key-plate is 3D printed. The keyboard features a QMK firmware implementation that allows for custom key mappings and macros. The project was based on the Scotto44 keyboard design.

Procedure Applied:

  • Designed and built a custom split mechanical keyboard using an RP2040 microcontroller.
  • Hand-soldered a logic matrix for key detection.
  • Created a laser-cut base and 3D printed key-plate for the keyboard.
  • Implemented QMK firmware for custom key mappings and macros.

Skills Applied:

  • Microcontroller programming (RP2040)
  • Keyboard firmware development (QMK)
  • Electronics design and prototyping
  • 3D modeling and printing for custom parts

Key Outcomes:

  • Successfully created a fully functional split keyboard with customizable layouts and works on any operating system
