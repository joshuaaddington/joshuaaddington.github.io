Line Following Robot

This project involved designing and building a line following robot using a PIC24 microcontroller. We were able to use PD control in conjunction with infrared sensors to achieve a robust line following behavior. The project was a great deep-dive into embedded systems as we started from first principles such as oscillators, bit registers, op-amp circuits, analog to digital conversion, and more.

Procedure Applied:

  • Designed and implemented PD control algorithms on a PIC24 microcontroller.
  • Integrated infrared sensors for real-time line detection.
  • Developed and tested op-amp circuits for sensor signal conditioning.

Skills Applied:

  • Embedded C programming
  • Microcontroller interfacing
  • Analog circuit design
  • Control systems theory

Key Outcomes:

  • Successfully developed a line following robot capable of navigating complex paths.
  • Gained hands-on experience with embedded systems and real-time control.
